# 04. AI Job Application Helper

**Resume Analyzer + Cover Letter Generator + Interview Prep**  
An AI tool to help with job applications, especially in AI/ML/Software Engineering roles.

### Features
- Resume analysis and suggestions
- Cover letter generation tailored to job descriptions
- Interview question preparation
- Ethiopian context (local job market tips)

### Tech Stack
- Hugging Face Transformers
- Gradio UI
- Prompt Engineering
- RAG for job-related knowledge

### How to Use
1. Upload your resume
2. Paste job description
3. Get personalized cover letter and suggestions
